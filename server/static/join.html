Join the HavnAI GPU Grid

Run this installer on your GPU machine. Replace <server> with your coordinator address if different.

curl -fsSL http://<server>:5001/static/install-node.sh | bash -s -- --server http://<server>:5001

If you have a join token, append --token <TOKEN>.

To disconnect later:

curl -X DELETE http://<server>:5001/unregister -H 'Content-Type: application/json' -d '{"node_id":"<your-hostname>"}'
