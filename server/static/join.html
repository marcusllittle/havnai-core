Join the HavnAI GPU Grid

Run this installer on your GPU machine. Replace <server> with your coordinator address if different.

curl -fsSL http://<server>:5001/installers/install-node.sh | bash -s -- --server http://<server>:5001

If you have a join token, append --token <TOKEN>. To prefill a wallet, add --wallet 0x....

Prerequisites

  • 64-bit Linux (Ubuntu/Debian/RHEL) or macOS 12+
  • Python 3.10+, curl, and a GPU driver/runtime (NVIDIA + CUDA recommended)
  • 12 GB+ NVIDIA GPU recommended for creator/video workloads (CPU nodes still supported)
  • EVM wallet address (simulated rewards in Alpha)

What happens next?

  1. Installer prepares ~/.havnai, Python venv, and the node client
  2. Configure WALLET / JOIN_TOKEN inside ~/.havnai/.env (or pass flags)
  3. Start the service (systemd or launchd) or run directly
  4. Monitor progress via dashboard and leaderboard

To disconnect later:

curl -X POST http://<server>:5001/disconnect -H 'Content-Type: application/json' -d '{"node_id":"<your-hostname>"}'
